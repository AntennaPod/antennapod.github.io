Hai algunhas opcións para copiar os teus datos ao teu teléfono novo, e cada método copia unha cantidade de datos definida:

## Exportar a base de datos

Exportar a base de datos de AntennaPod é un bo método se estás migrando a un teléfono ou tablet novos. Deberás de asegurarche de que tes a mesma versión de AntennaPod en ambos os dispositivos. Despois de importar a base de datos, todos os datos estarán reestablecidos; só necesitarás volver descargar os episodios.

## Sincronización

Sincronizar AntennaPod a través de *gpodder.net* ou a aplicación *gPodder Sync* para Nextcloud permíteche gardar e migrar algúns datos sobre os podcasts e episodios. Perderás os teus favoritos, a cola e os axustes da aplicación. Isto pode ser unha solución cando rota entre teléfono, tablet e/o computador. Por favor mira o [artigo sobre a sincronización](/documentation/xeral/synchronization) para instrucións.

## OPML

Os ficheiros OPML permítenche transferir as túas subscricións a outra aplicación de podcasts. Perderás calquera información relacionada cos podcast e os episodios (ex. que episodios escoitaches) e todos os axustes da aplicación. Para exportar o teu ficheiro, vai a `Axustes` » `Importar/Exportar` » `Exportación OPML`. Dependendo da versión de Android igual che pregunta onde queres gardar o ficheiro. Se se abre automaticamente unha *xanela de gardado* pon coidado en elixir un cartafol fóra do directorio de AntennaPod (ex., `Descargas` ou outro cartafol accesible), para que o ficheiro OPML non se elimine en caso que desinstales AntennaPod.

Se importas o arquivo en Antennapod noutro dispositivo, a aplicación mostrará unha lista de podcasts con casas de selección. Os podcasts que seleccións desa lista engadiranse ás túas subscricións existentes. Ningunha das túas subscricións existentes será afectada.
