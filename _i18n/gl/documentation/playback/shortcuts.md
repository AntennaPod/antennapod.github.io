Se tes un teclado físico ('hardware') conectado ao teu teléfono ou tablet, podes utilizar certas combinacións de teclas para controlar a reprodución en AntennaPod. O volume e as accións de (deas)enmudecer afectan a canle de audio para os medios (en lugar de, por exemplo, o das notificacións).

### Todas as pantallas (excepto o reprodutor de vídeo e os axustes)

| Tecla | Acción |
| --- | --- |
| `P` | Reproducir/Pausar |
| `J`/`A`/`,` | Rebobinar |
| `K`/`D`/`.` | Adiantar |
| `+`/`W` | Elevar o volume |
| `-`/`S` | Baixar o volume |
| `M` | Enmudecer/Desenmudecer (alternar) - só en Android 6 ou superior |

### Reprodutor de vídeo

| Tecla | Acción |
| --- | --- |
| `P`/`Espazo` | Reproducir/Pausar |
| `J`/`A`/`,` | Rebobinar |
| `K`/`D`/`.` | Adiantar |
| `F`/`Esc` | Saír do reprodutor de vídeo |
| `i` | Entrar no modo imaxe-en-imaxe |
| `0`-`9` | Ir ao 0-90% do vídeo |
| `+`/`W` | Elevar o volume |
| `-`/`S` | Baixar o volume |
| `M` | Enmudecer/Desenmudecer (alternar) - só en Android 6 ou superior |
