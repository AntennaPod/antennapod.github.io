Löysitkö kirjoitusvirheen tai jotain, joka kuulostaa oudolta? On aina mahdollista, että teimme virheen englanniksi (jolla teemme sovelluksen ja verkkosivuston) tai jotain ei ole käännetty oikein. Onneksi nämä ongelmat korjataan usein helposti.

# Haluatko ratkaista ongelman itse?

Jos ongelma on jollakin muulla kuin englannin kielellä, voit päivittää käännökset [Weblatessa](https://hosted.weblate.org/projects/antennapod/). Voit [katso lisätietoja Osallistu kehitykseen-osiossamme](/contribute/translate).

Onko ongelma **englanninkielisissä teksteissä**? Sitten sinun on tunnettava GitHub tai tutustuttava siihen. Voit lähettää vetopyynnön [sovellukselle](https://github.com/AntennaPod/AntennaPod/) tai [verkkosivustolle](https://github.com/AntennaPod/antennapod.github.io).

# Haluatko vain ilmoittaa meille asiasta?

Voit luoda viestin [keskustelupalstamme käännökset-osioon](https://forum.antennapod.org/c/translations/11). Tätä varten sinun on luotava tili (jos sinulla ei vielä ole) tai kirjauduttava sisään Googlen kautta.
