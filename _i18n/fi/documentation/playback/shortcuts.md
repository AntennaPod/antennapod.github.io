Jos puhelimeesi tai tablettiisi on liitetty fyysinen ("laitteisto") näppäimistö, voit käyttää tiettyjä pikanäppäimiä toiston ohjaamiseen AntennaPodissa. Äänenvoimakkuuden säätö ja mykistyksen poisto (tai poisto) vaikuttavat median äänikanavaan (eikä esimerkiksi ilmoitusten äänikanavaan).

### Kaikki näytöt (paitsi videosoitin ja asetukset)

| Näppäin | Toiminto |
| --- | --- |
| `P` | Toisto/tauko |
| `J`/`A`/`,` | Kelaa takaisin |
| `K`/`D`/`.` | Eteenpäinkelaus |
| `+`/`W` | Lisää äänenvoimakkuutta |
| `-`/`S` | Alenna äänenvoimakkuutta |
| `M` | Mykistä / poista mykistys (vaihda) – vain Android 6 ja uudemmat |

### Videosoitin

| Näppäin | Toiminto |
| --- | --- |
| `P`/`Välilyönti` | Toisto/tauko |
| `J`/`A`/`,` | Kelaa takaisin |
| `K`/`D`/`.` | Eteenpäinkelaus |
| `F`/`Esc` | Poistu videosoittimesta |
| `i` | Siirry kuva kuvassa -tilaan |
| `0`-`9` | Siirry videon kohtaan 0–90 % |
| `+`/`W` | Lisää äänenvoimakkuutta |
| `-`/`S` | Alenna äänenvoimakkuutta |
| `M` | Mykistä / poista mykistys (vaihda) – vain Android 6 ja uudemmat |
