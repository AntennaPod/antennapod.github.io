Há algumas maneiras de copiar seus dados para um novo celular, e cada método copia uma quantidade diferente de dados:

## Exportar banco de dados

Exportar o banco de dados do AntennaPod é uma ótima opção se você estiver migrando para um novo celular ou tablet. Certifique-se de que a mesma versão do AntennaPod esteja instalada em ambos os dispositivos. Depois de importar, todos os seus dados serão restaurados; você só precisará baixar novamente os episódios.

## Sincronização

Sincronizar o AntennaPod por meio do *gpodder.net* ou do app *gPodder Sync* para Nextcloud permite salvar e migrar parte dos dados de podcasts e episódios. Você perderá seus favoritos, fila e configurações do app. Essa pode ser uma boa solução ao alternar entre celular, tablet e/ou computador. Consulte o [artigo sobre sincronização](/documentation/general/synchronization) para ver as instruções.

## OPML

Arquivos OPML permitem transferir suas inscrições para outro aplicativo de podcasts. Você perderá todos os dados específicos dos podcasts ou dos episódios (p. ex., quais episódios já reproduzidos) e todas as configurações do app. Para exportar seu arquivo, vá para `Configurações` » `Importar/Exportar` » `Exportar OPML`. Dependendo da sua versão do Android, pode ser solicitado que você escolha onde salvar o arquivo. Se uma *caixa de diálogo de salvamento* abrir imediatamente, certifique-se de selecionar uma pasta fora do diretório do AntennaPod (p. ex., `Downloads` ou outra pasta visível), para que o arquivo OPML exportado não seja excluído caso você desinstale o AntennaPod.

Ao importar o arquivo no AntennaPod em outro dispositivo, o app exibirá uma lista de podcasts com caixas de seleção. Os podcasts que você marcar nessa lista serão adicionados às suas inscrições existentes. Nenhuma inscrição atual será afetada.
