Encontrou um erro de digitação ou algo que soa estranho? É sempre possível que tenhamos cometido um erro em inglês (idioma em que o app e o site são desenvolvidos) ou que algo não tenha sido traduzido corretamente. Felizmente, esses problemas costumam ser fáceis de corrigir.

# Você quer corrigir o erro por conta própria?

Se o problema estiver em outro idioma que não o inglês, você pode atualizar as traduções no [Weblate](https://hosted.weblate.org/projects/antennapod/). Você pode [encontrar mais informações em nossa seção Participe](/contribute/translate).

O erro está nos **textos em inglês**? Nesse caso, é preciso conhecer ou se familiarizar com o GitHub. Você pode enviar um Pull Request para o [app](https://github.com/AntennaPod/AntennaPod/) ou para o [site](https://github.com/AntennaPod/antennapod.github.io).

# Prefere apenas nos informar?

Você pode criar um tópico na [seção de traduções do nosso fórum](https://forum.antennapod.org/c/translations/11). Para isso, é necessário criar uma conta (caso ainda não tenha uma) ou simplesmente fazer login com o Google.
