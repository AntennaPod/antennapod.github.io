    Impossible de télécharger un épisode

    AntennaPod est une application de podcast décentralisée. Cela signifie qu’il n’y a pas de serveur central capable de tout savoir à propos de chaque podcast.

    Les fichiers multimédias proviennent donc directement du site web de l’éditeur. Si l’éditeur les supprime, AntennaPod ne peut plus les récupérer pour vous.

    Le gestionnaire de podcasts simple, flexible et open-source pour Android.
